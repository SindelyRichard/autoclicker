# Autoclicker for Hypixel Fishing

## Description
This Python script automatically fishes on the Hypixel Minecraft server.

## Library
You have to install pyautogui in order to run the script.
You have to install python and and pip.

## Installation

1. Clone this project.
2. Run it with the following command:
   ```
   python clicker.py
   ```
3.Stop it with Ctrl+c

## Result
In 10 minutes of fishing, I caught 40 fish on Hypixel Housing.
